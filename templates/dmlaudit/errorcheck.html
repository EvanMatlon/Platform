报错白皮书

  • ORA错误
  • MYSQL错误
  • ORA-00001 unique constraint (...) violated 违反（...）字段的唯一约束条件
    唯一性字段上出现重复数据导致
  • ORA-00903 invalid table name 无效表名
    表名前后有引号或者逗号等符号导致
  • ORA-00904 invalid identifier 标识符无效
    大部分情况下是由于引用了不存在的列名导致
  • ORA-00942 table or view does not exist 表或视图不存在
    数据库中没有相应表导致
  • ORA-01008 not all variables bound 并非所有变量都已绑定
    使用了绑定变量导致，数据订正语句中不允许使用绑定变量
  • ORA-01722 invalid number 无效数字
    对number类型的字段赋予了字符串型的数据导致
